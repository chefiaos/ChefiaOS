# Como usar `_decisoes/`

Esta pasta é a **memória histórica do negócio**. Tudo o que foi decidido, tudo o que funcionou, tudo o que falhou, tudo o que foi aprendido. É o ativo mais valioso do ChefiaOS — o que diferencia uma operação amadora de uma operação que aprende.

## O que salvar

- **Decisões estratégicas** — "vamos focar em orgânico antes de ads"
- **Resultados de experimentos** — "o Reel com gancho de dor teve 3x mais alcance"
- **O que funcionou** — "email com assunto curto abriu 40%"
- **O que NÃO funcionou** — "carrossel educacional não converteu"
- **Insights de usuários** — "usuários reclamam que a leitura é longa"
- **Mudanças de direção** — "pausei TikTok para focar no Instagram"
- **Números relevantes** — "CAC em maio: R$ 18"

## Onde salvar

1. **Sempre:** `[ANO]/[MM_mes]/LOG.md` — log mensal. Crie a pasta se não existir.
2. **Se for atemporal:** também em `aprendizados/[AREA].md`.
3. **Se for experimento:** crie um arquivo em `experimentos/` com base em `experimentos/TEMPLATE_EXPERIMENTO.md`.

## Formato obrigatório de cada entrada

```
## [DATA] — [TÍTULO]
**Tipo:** Decisão / Resultado / Aprendizado / Experimento
**Área:** Marketing / Vendas / Produto / Financeiro / Operações / Clientes
**Cargo relevante:** CEO / CMO / CFO / CTO / CPO / COO / CDO / CHRO

**O que aconteceu / foi decidido:**

**Por quê:**

**Resultado (se já souber):**

**Aprendizado:**

**Próximo passo:**
```

## Regra de ouro

**Nunca apague entradas antigas.** Decisão registrada permanece registrada, mesmo que depois você mude de ideia. Mudar de ideia é uma nova entrada. A história é o ativo.
