# Aprendizados — CLIENTES

Este arquivo guarda aprendizados **atemporais** sobre relação com clientes (onboarding, suporte, churn, feedback, retenção). Aqui ficam padrões que se repetem, princípios validados, regras que valem para qualquer mês.

Diferença para o LOG mensal: o LOG registra eventos no tempo; este arquivo registra **conclusões que sobrevivem ao tempo**.

## Quando adicionar aqui
- Quando o mesmo aprendizado aparecer 2-3 vezes em meses diferentes.
- Quando um experimento gerar um princípio aplicável a qualquer situação futura.
- Quando uma intuição virar regra após validação.

## Formato sugerido por aprendizado
```
### [TÍTULO CURTO DO PRINCÍPIO]
**Evidência:** [referências aos logs ou experimentos que sustentam]
**Princípio:** [a regra em 1-3 frases]
**Aplicação:** [como usar isso na prática]
**Exceções conhecidas:** [quando não vale]
```
