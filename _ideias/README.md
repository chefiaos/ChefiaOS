# `_ideias/` — Brain dump

Pasta de captura rápida. Ideias soltas, insights aleatórios, hipóteses não-validadas, intuições, sementes de projetos futuros.

## Diferença entre `_ideias/` e `_decisoes/`

| | `_ideias/` | `_decisoes/` |
|---|---|---|
| Status | Não-validada | Tomada / validada |
| Compromisso | Nenhum | Implícito |
| Pode esquecer? | Sim | Não |
| Vira ação? | Talvez | Sim |

## Quando uma ideia vira decisão

Quando você decidir testar, executar ou adotar uma ideia, **mova-a para `_decisoes/[ANO]/[MM_mes]/LOG.md`** com o formato padrão. A ideia pode permanecer aqui como histórico, mas a decisão registrada vira o documento canônico.

## Como capturar

- Crie arquivos soltos com nomes descritivos: `ideia_lead_magnet_quiz.md`, `feature_modo_offline.md`, etc.
- Não se preocupe com formato. O importante é capturar antes de esquecer.
- Revise mensalmente: o que ainda faz sentido? O que virou decisão? O que pode ser arquivado?

## Comando útil

Diga `/empresa ideia: [descrição]` para o sistema salvar automaticamente aqui.
